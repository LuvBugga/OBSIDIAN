(Folder and Directory are basically the same thing but for different types).
(Folder for GUI's (Graphical User Interface) and Directory for Terminal use).
("/" is the Root Directory)
1. pwd (Print Working Directory) : Prints the folder you are in.
2. ls (List) : Lists all the folders in the current folder you are in.
3. cd (Change Directory) : It helps you go into different directories.
   - cd .. (cd dot dot) : Takes you back in the previous directory you were in.
4. whoami : Tells you the username by which you are logged in.
5. clear : It clears the entire Terminal. (CTRL + L)
6. cat (Concatenate) : It shows you what is inside of a file (eg ls or pwd).
7. cp (Copy) : It copies a file into another file. 
8. sudo (Super Do) : Allows the user to perform an admin task.
9. rm (Remove) : It deletes files.
10. adduser : Adds a new user.
11. which : Tells us from where the command or file is used.
12. ps (Process Status) : Shows you the current ongoing processes. (Also your type of Shell)
13. id = Gives all the info. about your ID.
14. hostname = Tells you the Host name.
15. ifconfig = Shows you network stuff.
16. ip = Shows you more network stuff.
17. netstat = Shows you the status of your network.
18. ss (Session Status) : Shows you the status of your current session.
19. who = Tells you who else is logged in the system.
20. env = Environment Variables
21. lsblk (List Blocks) : Hard-drive Stuff
22. lsusb = Shows you if there are any USB"s plugged in
23. lsof = Lists all the open files
24. man = Shows you all the info about all the commands (Think of it like chatgpt but it only works for Linux commands)
25. -h or --help = Shows you about the command asked (HELP)
26. apropos = It kind of gives you all the related commands based on a keyword you give. (Eg: If you want to do something related to USB, write apropos usb - it will show you everything related to USB)
